### 공개키 암호화

![Untitled](./imgs/encryption-6.png)

- 암호화와 복호화에 사용하는 키가 서로 다른 기법
    - 암호화할 때에는 public key 를 사용
    - 복호화할 때에는 private key 를 사용
- public key 는 공개적으로 제공하여 사용자는 해당 키를 통해 암호화할 수 있지만 암호화한 데이터는 private key 를 가진 소유자만이 복호화할 수 있다.
